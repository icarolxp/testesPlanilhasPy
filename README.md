# Automação e Manipulação de Planilhas com Python

Bem-vindo ao meu laboratório de automação de dados! Este repositório foi criado para documentar meus exercicios no uso de Python para interagir com planilhas (Excel `.xlsx` e LibreOffice `.ods`).

O objetivo aqui é duplo: conhecer as principais bibliotecas do mercado através de excercicios isolados e aplicar esse conhecimento em projetos futuros mais estruturados, separando o aprendizado experimental da aplicação profissional.

##  Tecnologias e Ferramentas

O desenvolvimento é realizado em ambiente **Linux (Fedora)**, priorizando compatibilidade e ferramentas Open Source.

* **Linguagem:** Python 3
* **Análise de Dados:** Pandas
* **Estilização e Gráficos:** XlsxWriter
* **Edição de Arquivos:** OpenPyXL
* **Compatibilidade Linux:** ODFpy (suporte nativo ao LibreOffice)

##  Roteiro de Aprendizado (Roadmap)

### Fase 1: Domínio das Ferramentas (Testes Simples)
- [ ] **Pandas:** Leitura, escrita e filtragem eficiente de grandes volumes de dados (CSV/Excel).
- [ ] **XlsxWriter:** Geração de relatórios visuais com gráficos nativos e formatação condicional.
- [ ] **OpenPyXL:** Edição cirúrgica de planilhas e templates já existentes.
- [ ] **Integração Linux:** Testes de compatibilidade total com o formato `.ods` (LibreOffice).

### Fase 2: Aplicação Prática (Projetos Modulares)
- [ ] Desenvolvimento de uma aplicação modular.
- [ ] Implementação de interface ou CLI para interação com o usuário.
- [ ] Separação clara entre lógica de negócio e manipulação de arquivos.
